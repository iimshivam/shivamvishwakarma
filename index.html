Project Highlights

A network analysis project leveraging Python in Jupyter Notebook (Anaconda) forms the core of this work. It involves a deep dive into a large dataset from a multinational corporation, utilizing NetworkX, NumPy, and Matplotlib. The project focuses on constructing and visualizing complex networks, analyzing attributes like centrality and clustering, and generating histograms for detailed insights into organizational structures and relationships.

Network Analysis of Fuji Electric

This project is an in-depth analysis of Fuji Electric Co. Ltd., exploring its history, global expansion strategies, core technologies, and focus on sustainable solutions, particularly in the electric vehicle sector.

Climate Analysis

In this R project, CO2 data from NASA's climate website was analyzed using R language and RStudio. The script handles data importation, cleaning, and transformation, including renaming columns and converting dates. ggplot2 was utilized for data visualization, creating line plots to highlight monthly average CO2 levels and seasonal trends. Excel was used to address data inconsistencies or errors. This project showcases the integration of R programming and Excel in analyzing and visualizing climate data effectively.

Carbon Footprint pattern

This project employs R to analyze CO2 climate data, focusing on data transformation and visualization. It includes data cleaning and the use of graphical tools to highlight CO2 trends and seasonal patterns, showcasing R's strengths in handling and interpreting environmental data.

Microsoft POWER BI dashboard

This project involved creating a Power BI dashboard for sales data and COVID-19 vaccination visualization. The data, sourced from Kaggle, underwent an ETL (Extract, Transform, Load) process in Power BI. The dashboard features bars, maps, and graphs, designed for clear and understandable visual representation of the data. This approach effectively utilized Power BI's capabilities for interactive and insightful data visualization.

Statistical Analysis using JASP/SPSS

The dissertation involves statistical analysis using JASP, a free alternative to SPSS. It focuses on cross-cultural issues in operational management at Procter & Gamble (P&G). Primary data was collected, cleaned, transformed, and loaded using Microsoft Excel. The study provides insights into managing cross-cultural dynamics in multinational corporations, with a detailed case study on P&G. The research contributes valuable findings and discussions to the field of cross-cultural management in multinational settings.
